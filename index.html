📋 比赛规则

📝 参赛要求

  • • 参赛者需完成实名认证
  • • 支持个人或团队参赛（团队最多3人）
  • • 每位选手只能参加一支队伍
  • • 参赛者需同意并遵守比赛规则

💻 提交规范

  • • 代码需通过 Git 提交到指定仓库
  • • 每天最多提交 10 次
  • • 提交后系统自动评测，实时更新分数
  • • 以最高分数作为最终成绩

⚖️ 评分标准

  • • 正确性 (60%): 测试用例通过率
  • • 性能 (30%): 运行时间和内存占用
  • • 代码质量 (10%): 可读性和规范性
  • • 同分情况按提交时间排序

🚫 禁止行为

  • • 禁止抄袭他人代码或使用非法手段
  • • 禁止恶意攻击评测系统
  • • 禁止多账号参赛
  • • 违规者将取消参赛资格

❓ 常见问题

参赛选手需要创建私有github仓库，将机器人帐号@kernerl-competition-bot作为合作者加入仓库并按照指南配置仓库的webhook。配置完成后，选手每次 push 会自动触发评测。评测结果由自动更新至排行榜和提交记录评论区

评测系统运行在 Linux 环境下，配置为 MLU370（具体型号待备注，设备详细信息查询寒武纪官网），每道题目的时间限制为1分钟。

可以通过邮件联系我们，详见联系我们页面。工作日 24 小时内会回复。
